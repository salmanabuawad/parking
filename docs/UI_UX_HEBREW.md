# Hebrew UI/UX Notes

## Layout
- medium-sized video player
- evidence / decision sidebar always visible
- screenshot strip below player
- debug panels collapsed by default

## Hebrew statuses
- confirmed: מאומת
- very_likely: סבירות גבוהה
- possible: אפשרי
- not_100_percent_sure: לא ודאי ב־100%
- rejected: נדחה

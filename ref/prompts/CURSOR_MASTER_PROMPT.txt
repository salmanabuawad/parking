Build the final masterpiece version of this app as a modular Python codebase.

Primary goal:
Process street video and blur everything except the validated Israeli target plate.

The target plate must be:
1. detected in the frame
2. tracked across frames
3. read with OCR
4. aggregated over time
5. validated against the Gov.il private/commercial vehicle registry dataset

Then:
6. classify the plate format using Israeli standard dimensions
7. optionally detect red/white curb candidates
8. save output video, debug frames, and JSON results

Architecture rules:
- keep app.py thin
- put config in config.py
- use interface-based detector design
- support hsv and yolo detector backends
- OCR must be optional and failure-safe
- OCR results must go through ocr_vote + registry validation
- keep debug output separate from core logic
- keep code ready for FastAPI integration

Required modules:
- config.py
- video_io.py
- plate_detector.py
- tracker.py
- ocr_reader.py
- ocr_vote.py
- registry_lookup.py
- plate_format.py
- curb_detector.py
- distance.py
- debug.py
- blur_pipeline.py

Gov data logic:
- normalize plate numbers to digits only
- accept only 7-digit or 8-digit strings
- validate only against registry matches
- expose manufacturer/model/year in results

Plate dimensions:
- private_long = 52x12 cm
- private_rect = 32x16 cm
- motorcycle = 17x16 cm
- scooter = 17x12 cm

Deliverables:
- runnable CLI
- tests
- requirements.txt
- clean docstrings
- debug mode
- JSON results schema compliance
